The Biology Trust Gap / 生物 AI 的可信缺口

生物 AI 不缺模型，缺的是预测背后的可信上下文和可复现性。一个虚拟细胞的扰动预测、一个蛋白质功能判断背后，模型版本（scGPT 基座 + RL 微调）、数据来源（细胞 / 蛋白质 / 临床）、评测脚本、随机种子和运行环境往往不可验证；一个生物学假设是否合理、由哪位专家 curate、被谁质疑和修订，也很难追踪。这正是高校合作与药企采用最担心的两件事：结果能不能复现，数据贡献能不能被确权和结算。

Setu 解决的不是“把预测上链”，而是把生物世界模型的执行、专家判断、形式化验证与数据贡献结算，组织成可验证、可解释的因果历史。

The Four Pillars · 生物世界模型的四大支柱

PILLAR 01

Verified Bio Benchmark

以 Solver 执行、TEE 执行证明与 Sifakis 形式化验证为生产级目标，把虚拟细胞预测、扰动响应、蛋白质性能的自报式跑分，转化为可审计、可复核、可引用的 benchmark receipt，服务论文发表与药企靶点验证。

PILLAR 02

Model & Data Passport

把生物基座模型（scGPT fork + RL 微调）与细胞 / 蛋白质 / 临床数据的身份、版本、哈希与收益分配逻辑记录在链上，真实权重与数据集保留在链下。这是与高校 / 药企合作时 ownership 建立的核心：合作中拿到的数据可被确权，用于重新微调升级模型。

PILLAR 03

Verified Deliberation + Formal Verification

记录一个生物学假设如何被提出、质疑、修订和支持。它不裁决机制真假，而是确认审议过程的因果历史，让一个靶点 / 通路判断在多专家与 Agent 挑战下逐步积累鲁棒性。配合 Sifakis 形式化验证，可对模型输出的关键性质（如因果顺序、约束一致性）做可检查的保证，而不只是黑盒预测。

PILLAR 04

Solver Market & Data DAO

Solver 节点执行虚拟细胞 / 蛋白质推理与 RL 微调任务，未来可通过 TEE remote attestation + measurement allowlist 验证执行环境；目标是通过 Setu 原生结算自动分配任务费用与数据贡献收益，支撑面向全球数据贡献者的 Data DAO 生意。

THE MAIN LOOP · DIRECTIONAL

Verified Bio Optimization Loop
虚拟细胞世界模型的端到端可信迭代（目标形态）

四个支柱不是孤立能力，而是生物世界模型持续变准的同一条主线。作为目标形态，benchmark 不是一次性跑分，而是持续过程：让虚拟细胞 / 蛋白质模型在可版本化的 in-silico 任务环境中生成预测，由客观指标（与 wet-lab / 公开基准的一致性）衡量，再引入专家与 Agent 的生物学判断进行解释；据此修订模型、数据集或任务规格，更新后的模型再回到预测，形成闭环。Setu 的方向是用 logic clock / 时序逻辑把这条闭环串成可验证的因果历史：什么必须先于什么，谁的生物判断基于谁的预测。

这正是“基座模型 + RL 微调”的可信化。Setu 把生物模型迭代中的 reward 拆成可验证的客观指标（Measure）+ 可追踪的专家判断（Judge），而不是一个不可解释的标量。这让每一轮虚拟细胞模型的升级都可解释、可复核、可结算给数据与算力贡献者。

01 · Rollout

Solver 让虚拟细胞 / 蛋白质模型在受控、可版本化的 in-silico 任务环境中生成预测；模型与数据版本由 passport 锰定，输入与环境被提交。

monitoring

02 · Measure

基于客观指标（与 wet-lab / 公开基准的一致性、扰动预测准确率等）生成可审计、可复核的 receipt；TEE attestation 接入后升级为硬件级 verified receipt。

rate_review

03 · Judge

生物学家与 Agent 的主观判断介入：质疑、支持、解释预测背后的生物学机制与语境。

tune

04 · Update

基于客观指标与专家判断，对基座模型做 RL 微调、更新数据集或任务规格。

replay

05 · Re-run & Settle

更新后的模型回到预测；每轮数据 / 算力 / 评测贡献按 receipt 自动结算，形成可追踪、可复核的迭代。

rollout → measure → judge → update → re-run → settle · RL loop ordered by logic clock / temporal logic
